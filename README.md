## 📱 Published App

### Charadefy

**Charadefy** is a mobile charades game built with **React Native**, available on both **App Store** and **Google Play**.

The app allows users to play a fun guessing game using categories, timed rounds, sounds, score tracking and mobile interactions designed for a smooth party-game experience.

- Built with React Native
- Available for iOS and Android
- Published on App Store and Google Play
- Uses device interaction and mobile-oriented UX
- Includes game configuration, countdown, scoring and category selection
